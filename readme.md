# 🎨 JavaScript Color Changer

A simple mini project built using **HTML, CSS, and JavaScript** that allows users to change the background color of the webpage by clicking buttons.

---

## 🚀 Features
- 🎯 Change background color with a single click  
- 🎨 Multiple color options (buttons)  
- ⚡ Instant UI update using JavaScript DOM  
- 🧠 Beginner-friendly project  

---

## 🛠️ Tech Used
- HTML  
- CSS  
- JavaScript (DOM Manipulation)  

---

## 📂 How It Works
- Each button has a specific color  
- When a button is clicked, JavaScript:
  - Detects the click event  
  - Updates the background color of the page  

---

## 💡 Learning Outcome
- Understanding of **DOM manipulation**  
- Handling **click events**  
- Basics of **JavaScript interactivity**
